### Technologies

* Technologies: **Web Audio API, Webpack**

## Installing

**Firstly** clone the project.

```git
git clone https://github.com/l1ve4code/3d-music-visualizer-v2.git
```

**Secondly** install dependencies
```cmd
npm install
```

**Thirdly** run project

### For watch or development

```cmd
npm run dev
```

### For build

```cmd
npm run build
```
